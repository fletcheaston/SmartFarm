Raspberry Pi Ueful Commands

Shutdown pi:	
sudo poweroff

serial port:
/dev/ttyUSB0

installing applications over internet:
sudo apt-get install <name of software>

open folder:
cd folder_name
cd /opt/vc/src/hello_pi

folder back:
cd ..

Remote Desktop:
vncserver :1

Serial from arduino

find location of usb device(plug in, then remove to see what changed in list):
ls /dev/tty*

Create python script file(in this file change the location of '/dev/ttyUSB0' to the device):
sudo nano hello_world.py
-----CODE-----(don't include the ---'s)

import serial
ser = serial.Serial('/dev/ttyUSB0',57600)
while True:
        print ser.readline()

-------------
Run python script:
sudo python hello_world.py
